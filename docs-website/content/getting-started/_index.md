ClawIDE is a single Go binary that gives you a web-based IDE for managing multiple Claude Code sessions. This section covers everything you need to go from zero to a running instance.

## What You'll Need

- **Go 1.24+** — for building from source
- **Node.js** — for compiling frontend assets
- **tmux** — terminal multiplexer used as the session backend
- **Docker** (optional) — for Docker integration features or running ClawIDE in a container
